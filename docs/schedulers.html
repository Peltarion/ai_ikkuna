schedulers module¶

class schedulers.FunctionScheduler(optimizer, lr_fn)[source]¶

Bases: object

Scheduler class. This is a ripoff from torch.optim.lr_scheduler._LRScheduler because PyTorch learning rate schedulers only support a resolution of epochs. The need for changing the learning rate on a batch basis came about when reproducing the ResNet paper, as they change the learning rate at specific time steps.
