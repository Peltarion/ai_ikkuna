ikkuna.models.alexnet module¶

This module defines AlexNetMini, a reduced version of AlexNet. Adapted from torchvision.models.alexnet.

class ikkuna.models.alexnet.AlexNetMini(input_shape, num_classes=1000, exporter=None)¶

Bases: torch.nn.modules.module.Module

Reduced AlexNet (basically just a few conv layers with relu and max-pooling) which attempts to adapt to arbitrary input sizes, provided they are large enough to survive the strides and conv cutoffs.
