class ikkuna.visualization.backend.UpdatableHistogram(figure, subplot_conf=111, title='', max_hists=10, basecolor=(0.7803921568627451, 0.25098039215686274, 0.09411764705882353))[source]¶

Bases: object

A utility class to wrap a single histogram subplot of a figure which can display up to a number of histograms. New data displaces old one. This emulates tensorboard histogram traces with matplotlib.

replot()[source]¶

Recompute all visible plots. This will remove all current plots and simply replot self._hists.

Note

This is not an inefficient way to update the plot; the proper ways to do this is to simply move all current histograms one step backwards, unless the x axis changes.
