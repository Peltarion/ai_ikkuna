main program¶

This module contains functions and classes for simplifying the training of ANN classifiers. It accepts the following arguments:

usage: main.py [-h] -m {AlexNetMini,DenseNet} -d
               {MNIST,FashionMNIST,CIFAR10,CIFAR100} [-b BATCH_SIZE]
               [-e EPOCHS] [-o OPTIMIZER] [-a RATIO_AVERAGE] [-s SUBSAMPLE]
               [-v {tb,mpl}] [-V] [--spectral-norm]
               [--histogram [HISTOGRAM [HISTOGRAM ...]]]
               [--ratio [RATIO [RATIO ...]]] [--test-accuracy]
               [--train-accuracy] [--depth DEPTH]

Named Arguments¶

-m, --model

Possible choices: AlexNetMini, DenseNet

Model class to train

-d, --dataset

Possible choices: MNIST, FashionMNIST, CIFAR10, CIFAR100

Dataset to train on

-b, --batch-size
 Default: 128
-e, --epochs Default: 10
-o, --optimizer
 

Optimizer to use

Default: “Adam”

-a, --ratio-average
 

Number of ratios to average for stability (currently unused)

Default: 10

-s, --subsample
 

Number of batches to ignore between updates

Default: 1

-v, --visualisation
 

Possible choices: tb, mpl

Visualisation backend to use.

Default: “tb”

-V, --verbose

Print training progress

Default: False

--spectral-norm
 

Use spectral norm subscriber on weights

Default: False

--histogram Use histogram subscriber(s)
--ratio Use ratio subscriber(s)
--test-accuracy
 

Use test set accuracy subscriber

Default: False

--train-accuracy
 

Use train accuracy subscriber

Default: False

--depth

Depth to which to add modules

Default: -1
