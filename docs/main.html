main module¶

This module contains functions and classes for simplifying the training of ANN classifiers. It accepts the following arguments:

usage: main.py [-h] -m {AlexNetMini,DenseNet} -d
               {MNIST,FashionMNIST,CIFAR10,CIFAR100} [-b BATCH_SIZE]
               [-e EPOCHS] [-o OPTIMIZER]

Named Arguments¶

-m, --model Possible choices: AlexNetMini, DenseNet
-d, --dataset Possible choices: MNIST, FashionMNIST, CIFAR10, CIFAR100
-b, --batch-size
 Default: 128
-e, --epochs Default: 10
-o, --optimizer
 Default: “Adam”
